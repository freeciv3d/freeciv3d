<h1>The Freeciv 3D Online Manual</h1>

Freeciv 3D Fciv.net is a Free and Open Source empire-building strategy game inspired by the
history of human civilization.
<br><br>
Fciv.net / Freeciv-web is free and open source software. The Freeciv C server is released under the GNU General Public License while the Fciv.net (Freeciv-web) client is released under the GNU Affero General Public License
